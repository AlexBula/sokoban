#### Symbols Meaning:
* @ - player
* $ - box
* \# - wall
* . - storage 
* (space) \- floor

#### Additional informations 
Steering via WSAD keys.  
Undo last move - U  
Reset the whole game - R   

#### Running

Simply type `runhaskell sokoban.hs` in the terminal!
